# 🏦 ATM Simulator Project

A complete **ATM Simulator System** developed using **C++** with an additional **Modern Website-Based User Interface** built using **HTML, CSS, and JavaScript**.

This project simulates the basic functionality of an Automated Teller Machine (ATM) and demonstrates real-world banking operations such as login authentication, balance inquiry, money deposit, withdrawal, money transfer, and transaction history management.

# 📖 Project Introduction

The ATM Simulator Project is a software application that mimics the operations performed in a real ATM machine.

The project has two parts:

## 1️⃣ C++ Console Application
Implements the actual ATM logic using:
- Object-Oriented Programming
- Classes and Objects
- File Handling
- Functions
- Modular Programming

## 2️⃣ Website User Interface
A modern frontend interface that visually represents ATM operations in a professional banking-style dashboard.

The website improves:
- User experience
- Project presentation
- Interface design
- Portfolio quality

# 🧠 OOP Concepts Used

The project demonstrates several important OOP concepts:

Used to model ATM accounts and operations.

## 🔹 Constructors

# ▶️ Running The C++ ATM Simulator

# Step 1 — Open Terminal

Navigate to project directory:

```bash
cd ATM-Simulator
```

---

# Step 2 — Compile The Program

```bash
g++ main.cpp ATM.cpp Account.cpp -o atm.exe
```

---

# Step 3 — Run The Program

```bash
./atm.exe
```
